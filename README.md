This monorepo provides the following TypeScript LSP plugins:

- [no-private-imports](/liqvidjs/editor-plugins/tree/main/packages/no-private-imports): allows restricting imports based on `@package`/`@private` tags. Corresponds to the Biome rule [`no-private-imports`](https://biomejs.dev/linter/rules/no-private-imports/)

- [no-restricted-imports](/liqvidjs/editor-plugins/tree/main/packages/no-restricted-imports): allows restricting imports from a configured list of packages. Corresponds to the Biome rule [`no-restricted-imports`](https://biomejs.dev/linter/rules/no-restricted-imports/)
